# 直接排序，然后交换位置

将数组用任何可以的方式进行排序，然后成为有序数组后，从第二个开始两两交换
* 第二个小于等于第三个，交换后：**第二个大于等于第三个，肯定也大于等于第一个**
* 然后，第四个和第五个交换后：**五四无论怎么交换都大于等于第三个，所以，大于关系确定，接下来的小于关系通过交换第四个和第五个得到保障**
* 同理，第六和第七，本身就大于左侧的所有元素，所以，将6/7交换满足小于关系 即可。
